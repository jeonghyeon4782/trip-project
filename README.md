# "여행지를 선택하는 과정을 즐길 수 있는 콘텐츠를 제공하여 재미와 유익함을 주고 싶습니다!"

## 어떤 사람들을 위한 서비스일까요?
- 여행지를 선택하는 데 어려움이 있는 사람들
- 여행지를 다른 사람들에게 공유하고 추천해주고 싶은 사람들

## 👊 주요 기능

👤 **회원**
- 회원가입, 탈퇴
- 로그인, 로그아웃
- 소셜 로그인 (Kakao, Naver)
- 내가 작성한 리뷰 조회

🎲 **부루 마블**
- 지역 선택 후 랜덤으로 보드판 생성
- 주사위를 굴려서 위치 이동
- 무인도, 황금 열쇠, 우주 여행
- 관광지 도착 후 관광지 정보 조회

🍀 **커뮤니티**
- 게시글 작성, 수정, 조회, 삭제
- Best 게시글 조회
- 게시글에 대한 댓글 작성 및 수정, 삭
